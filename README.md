# Claude Repertoire

A curated collection of Claude Code skills and guides.

## Skills

Each directory under [`skills/`](skills/) is a standalone, self-documenting
skill — install one, or install them all.

## Docs

[`docs/`](docs/) holds guides and reference material for Claude Code
workflows and conventions.

## Install

```bash
git clone https://github.com/FrozenYty/claude-repertoire.git
ln -s "$PWD"/claude-repertoire/skills/* ~/.claude/skills/
```

## Contribute

Skills follow a shared structure: `SKILL.md`, `README.md`, `CONTRIBUTING.md`,
`CHANGELOG.md`. See [CONTRIBUTING.md](CONTRIBUTING.md) for the full spec.
