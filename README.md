# Claude Repertoire

Claude Code skills and guides, one `git clone` away.

```bash
git clone https://github.com/FrozenYty/claude-repertoire.git
ln -s "$PWD"/claude-repertoire/skills/* ~/.claude/skills/
```

Each directory under [`skills/`](skills/) is self-contained —
pick what you need, or install everything at once.

Guides and references live in [`docs/`](docs/).
